# Maze Runner — Java Desktop Game

Maze Runner is a Java-based 2D desktop game developed using Java Swing and AWT.  
The game features dynamic maze generation, player movement, timer-based gameplay, sound effects, and interactive UI screens.

The project demonstrates the use of Object-Oriented Programming, event handling, graphics rendering, and maze generation algorithms in Java.

---

## Features

- Dynamic maze generation
- Keyboard-controlled player movement
- Timer-based gameplay
- Win/Lose game states
- Sound effects and background music
- Collision detection
- Interactive UI using Java Swing
- CardLayout-based screen management

---

## Technologies Used

- Java
- Java Swing
- Java AWT
- OOP Concepts
- Event Handling
- DFS / Recursive Backtracking Algorithm

## How to Run

### Clone Repository

```bash
git clone https://github.com/your-username/Maze-Runner.git
```

### Open Project Folder

```bash
cd Maze-Runner
```

### Compile the Project

```bash
javac *.java
```

### Run the Project

```bash
java Main
```
